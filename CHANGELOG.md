## [1.0.0] — 2026-04-10

### Added
- Initial release of personal website
- Single-page layout with sections: Hero, About, Skills, Experience, Projects, Writing, Contact
- Dark / light mode toggle with OS preference detection and `localStorage` persistence
- Hero section with animated terminal window and typewriter cycling text
- About section with profile photo, bio, and animated stat counters (20+ years, 10+ projects, 2 languages)
- Skills section with 5 category cards and tech logos from Simple Icons CDN
- Experience timeline (Expedia Group — Engineering Manager)
- Projects grid: BULL (featured), BULL Tutorial, AWS S3 Downloader
- Writing section with article cards: DZone and InfoQ publications
- Contact section with Formspree form and copy-to-clipboard email
- Sticky frosted-glass navigation with active section highlighting
- Scroll reveal animations via `IntersectionObserver`
- Fully responsive layout — mobile hamburger menu included
- SEO: meta tags, Open Graph, Twitter Card, JSON-LD structured data
- `robots.txt`, `sitemap.xml`, `favicon.svg`
- Downloadable CV (`assets/resume.pdf`)
